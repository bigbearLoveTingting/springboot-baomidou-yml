星空 v1.00框架介绍

欢迎使用星空 v1.00框架。
星空 v1.00框架主要使用了以下组件组合而成，目的是打造一个快速开发API开发工具。


技术栈选型

  • spring boot 2.1.4.RELEASE
  • mybatis
  • mybatis pagehelper
  • redis
  • shiro
  • beetl
  • activemq

开发环境准备

  • 安装 JAVA
  • 安装 Maven
  • 安装 Eclipse
  • 安装 Svn
  • 安装 Redis
  • 安装 Navicat
  • 安装 Xshell
  • 安装 ActiveMq
